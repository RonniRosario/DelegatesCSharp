# DelegatesCSharp

Proposito del proyecto:

Su proposito es aprender la implementacion de los delegados y de las clases o metodos genericos, mediante la practica de hacer las diferentes operaciones matematicas basicas (suma, resta, multiplicacion y division), tambien usando las excepciones.

Excepciones que maneja:

Tenemos DivideByZeroException, esta impide que a la hora de que queramos realizar una operacion con el cero como numerador no nos lo permita.

Tenemos InvalidOperationException, la cual sirve para cuando se falla la invocacion de un metodo que su motivo de fallo sea distinto de razones de argumentos invalidos.

Tenemos Exception, que es una expresion generalizada para cualquier problema que pueda presentarse a la hora de ejecutarse el programa.

Como correr el programa:

Despues de presionar el boton verde para correr el proyecto, guiandonos del menu, introducimos el tipo de numero con el que trabajaremos, ya sea entero, decimal o double, luego de esto introducimos que operacion realizaremos y cuantos numeros añadiremos a la lista, luego de haber introducido los numeros correctamente, el programa nos devolvera el resultado.
